PLC-Integrated ROS Simulation for Smart Factory Robotics (2020)

This work was part of the project “Development of Integrated Control Technology supporting both control of 4 robots and PLCopen standard-based PLC language (5 types) for SME’s smart factory”, funded by South Korea’s Ministry of Trade, Industry, and Energy (MOTIE). The project aimed to create a standardized and efficient control system for industrial automation, enabling SMEs to integrate robots with existing PLC-based infrastructure seamlessly.

Project Overview

The core objective of this project was to bridge the gap between traditional PLC-based industrial automation and modern robotic simulation environments. Many SMEs rely on PLC controllers for automation but lack cost-effective methods for testing and validating robotic workflows before physical deployment. To address this, I developed an integration pipeline between PLC-controlled systems and Gazebo, an open-source robotics simulator, allowing real-time validation of industrial automation processes in a virtual environment. This system enabled manufacturers to test PLC logic with simulated robots, reducing downtime and minimizing the risk of errors in actual production lines.

My primary role in the project involved developing a middleware layer that allowed PLCs to communicate with simulated robots in Gazebo. By leveraging ROS (Robot Operating System) and an industrial communication protocol, I enabled bidirectional data exchange between the PLC and the robotic simulation, ensuring that sensor data, actuator commands, and task execution sequences were accurately mirrored in both environments. This integration allowed engineers to simulate multi-robot coordination, test task allocation strategies, and validate automation logic without the need for expensive physical prototypes.

Additionally, I worked on synchronizing robot motion with PLC instructions, ensuring that the Gazebo simulation adhered to real-world constraints such as motion timing, collision avoidance, and multi-robot cooperation. To enhance the system’s usability, I implemented a visualization tool that displayed PLC execution states alongside robot movements, helping engineers debug and optimize their automation programs effectively. This work ultimately streamlined the deployment of robotic automation in smart factories, enabling SMEs to transition more smoothly into Industry 4.0 with lower costs and reduced implementation risks.
