Automated Vertical Farm And Control System (2023)

Steering the proof of concept for urban vertical farms in Seoul, I led the design and implementation of a cost-effective, IoT-enabled farming system. This project aimed to demonstrate the feasibility of automated vertical farming in dense urban environments.

Project Overview

The project involved developing an automated farm controller using ESP32 and Arduino, implementing nutrient dosing and climate control with PID regulation, and deploying a cloud-based monitoring system using AWS IoT and MQTT for remote farm management.

A key achievement of this project was transforming a 700-square-foot basement in Seongsu-dong into a high-yield urban farm. Despite challenging conditions, the system successfully produced 1,200 fresh greens in just one month.

This project was a low-cost solution designed to address urban food production challenges by leveraging automation, microcontroller-based sensing, and real-time cloud monitoring. It served as a proof of concept for sustainable urban agriculture using affordable, scalable technology.
