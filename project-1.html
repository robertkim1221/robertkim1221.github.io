Vision-Language Task Planning for Furniture Assembly Robots (2025)

Developing a vision-language task planning system that enables robots to understand and execute furniture assembly using multimodal learning. By integrating video foundation models and large language models, the system extracts structured knowledge from human demonstrations to generate action plans for robotic execution.

Project Overview

Humans intuitively understand how to manipulate objects and perform tasks, but translating this intuition into adaptable, zero-shot robotic execution remains a challenge. This project explores how vision-language models and multimodal retrieval-augmented generation (RAG) can be leveraged to build structured knowledge representations for robotic task planning.

Focusing on IKEA furniture assembly, I am developing a multi-layered pipeline that decomposes high-level task understanding into robot-executable actions:

1. Multimodal RAG processes IKEA manuals to form a high-dimensional knowledge graph of assembly steps. 2. Video foundation models (VLMs) extract mid-level actions from human demonstrations. 3. Robot-grounded action primitives are inferred to generate structured, event-driven execution plans.

By structuring a deep knowledge representation of furniture assembly, this approach aims to bridge high-level task reasoning with low-level robotic execution, enabling more adaptive and semantically aware robot decision-making.

This system is built using state-of-the-art AI models to bridge high-level task reasoning with robot execution. A multimodal RAG pipeline powered by ColiPaLi vision retrieval extracts relevant assembly steps from IKEA manuals, forming a structured knowledge representation. InternVideo2, a video foundation model, is used to decompose human demonstrations into mid-level actions, while GPT-0.1 performs chain-of-thought reasoning to parameterize and ground action primitives into robot-executable commands. Together, these components create a context-aware decision-making system, enabling robots to infer and execute tasks with improved adaptability.
