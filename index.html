Projects A collection of robotics projects showcasing my work in motion planning, perception, and intelligent systems.

Vision-Language Task Planning for Furniture Assembly Robots (2025)

Developing a multi-layered knowledge representation system for robotic furniture assembly. Using a multi-modal RAG pipeline, this system extracts, decomposes, and stores IKEA assembly steps from human demonstration videos, enabling semantic understanding of physical interactions. The goal is to build a foundational robotic system augmented by domain-specific memory for reasoning and inference in similar tasks.

Operating Room Logistics Robot (2024)

Developed a robotic system for automating hospital logistics using the Fetch mobile manipulator. The system autonomously delivers medical supplies to operating rooms and restocks storage shelves, reducing staff workload and maintaining sterility. Built on a hardware-agnostic ROS2 software stack, it enables safe navigation in cluttered hospital environments while integrating seamlessly with human workflows. By automating critical logistics tasks, this system improves efficiency in high-pressure surgical settings.

Automated Vertical Farm And Control System (2023)

Led the development of a 700 sq. ft. automated vertical farm in downtown Seoul, producing 2,000 lettuce heads per month. Designed and constructed the farm infrastructure, built an IoT-enabled nutrient dosing controller, and developed a farm server for real-time monitoring and data management. This system optimized resource efficiency and demonstrated scalable urban farming solutions.

Case Study

Vision-Based Defect Detection for Solar Panel Maintenance Robots (2021)

Designed and built a vision module for an autonomous solar panel maintenance robot, integrating IR thermal cameras and CMOS image sensors to detect panel defects. Deployed at Korea East-West Power Plant, the system enabled early fault detection, reducing manual inspection efforts and improving operational efficiency.

Case Study

PLC-Integrated ROS Simulation for Smart Factory Robotics (2020)

Developed a PLC-integrated ROS simulation platform for verifying and validating PLC programs in smart factory robotics. Leveraging open-source simulators and ROS, the system enabled realistic testing of industrial automation workflows, reducing deployment risks and improving reliability in manufacturing environments.

Case Study

LiDAR-Based Localization for Above-Ground Tank Inspection Robots (2019)

Developed a LiDAR-based localization system for a mobile robot inspecting above-ground storage tanks. Using 2D LiDAR, the system achieved localization accuracy within 5 cm, enabling precise mapping and navigation around tank structures for inspection and maintenance applications.

Case Study

Dynamics Simulation of a Parallel Robot for Wind Turbine Inspection (2018)

Formulated and simulated the dynamics of a parallel robot’s end effector for NDT inspection of wind turbine blades. Using MATLAB, modeled stick-slip behavior to inform the design of a low-level controller, improving stability and precision during inspection tasks.
