Operating Room Logistics Robot (2024)

Hospitals rely on efficient logistics to maintain sterile, organized operating rooms, but manual restocking and retrieval of medical supplies place a heavy burden on staff. The Operating Room Bot (ORB) system is designed to automate these tasks using a robotic mobile manipulator built on the Fetch robot platform. ORB leverages autonomous navigation, perception-driven manipulation, and task-specific planning to deliver supplies during surgeries and restock shelves with minimal human intervention.

Project Overview

The Operating Room Bot (ORB) is a robotic mobile manipulator designed to automate hospital logistics, particularly in operating rooms. By autonomously retrieving and delivering medical supplies and restocking shelves, ORB enhances operational efficiency and allows hospital staff to focus on patient care. Built on the Fetch robot platform, the system integrates motion planning, perception, and high-level task execution to navigate and manipulate objects safely in dynamic hospital environments.

ORB is developed using a ROS2-based software stack, enabling seamless integration with hospital workflows. It features advanced autonomous navigation, perception-based object manipulation, and real-time decision-making to ensure reliability in high-stakes medical environments. Through intelligent automation, ORB reduces human workload, minimizes disruptions in sterile spaces, and optimizes hospital resource management.
